Guts Character Model from Berserk

Project Overview

This project showcases a personal project where I created a 3D model of Guts, the iconic character from the manga series Berserk. This was an ambitious challenge, as I had never created a realistic 3D character model before. My goal was to learn and implement the entire 3D character art pipeline, from initial sculpting to final rendering.

Process

  • Base Mesh Creation: Started with a basic human mesh to establish proportions and overall shape.
  • Sculpting: Using Blender's sculpting tools, meticulously sculpted the body and facial features, paying close attention to muscle structure and anatomical accuracy.
  • Armor and Weapons: Each piece of armor and Guts' iconic Dragonslayer sword was separately modeled using hard surface modeling techniques.
  • Retopology: Created a clean, low-poly retopology suitable for animation.
  • Rigging & Weight Painting: Rigged the character ensuring a functional skeleton that allows for a full range of motion.
  • Texturing: In Substance Painter, created realistic textures for the skin, armor, and weapon.

Tools Used

  • Blender - Modeling, Sculpting, Rigging, Rendering
  • Adobe Substance 3D Painter - Texturing

Project Duration

The entire project took approximately 3 months of full-time dedication, involving extensive trial and error, learning new software, and overcoming numerous challenges.

Dwarf Character for Fantasy Game

Project Overview

I was asked to create a game-ready Dwarf character for a university club's fantasy-themed game. Given a detailed concept, I completed the model in just two weeks, leveraging my experience from the previous Guts project.

Process

  • Concept & Sculpting: Quickly blocked out the character and sculpted key features like the beard and armor details.
  • Hair Creation: Used hair cards for the beard and hair, ensuring the model remained optimized for real-time use.
  • Rigging & Weight Painting: Rigged the character for smooth animations, paying special attention to areas like the beard and armor.
  • Texturing: Created detailed textures in Substance Painter, focusing on realistic materials and wear.

The character was completed efficiently, meeting game-ready standards. This project enhanced my skills in creating optimized models for game environments.
