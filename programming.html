Luden Engine

GitHub

Luden Engine is a lightweight 2D game engine built from scratch in C++. It features a fully custom Entity-Component System (ECS), an ImGui powered editor inspired by modern engines like Unity and Unreal. The engine focuses on flexibility, performance, and data-oriented design.

Entity-Component System (ECS)

At the core of Luden Engine is a custom ECS system designed for both speed and flexibility. Entities act as lightweight IDs, while components are stored in pools managed through templates and tuples.

  • Fast iteration over large numbers of entities
  • Cache-friendly data access
  • Clean separation of logic and data

Coop TPS Base Defense Game

Play Now!

The inspiration for developing this game came after playing Lethal Company. Its engaging co-op gameplay and the thrill of working together with friends to survive sparked a desire to create my own cooperative experience.

Gameplay Mechanics

Players collaborate to defend designated entry points of the base against waves of zombies. Each player shares the same abilities, focusing on shooting and resource management. Communication and teamwork are vital for surviving increasingly challenging assaults.

Networking

The game employs a server-client architecture where one player hosts the game as the server while others connect as clients. This setup enables replication and synchronization of game states across all players.

Motion Matching

Motion matching technology is employed to create smooth and realistic character animations. I used chooser tables for motion matching, which significantly simplified the debugging process.

AI Development

The zombie AI is created using Behavior Trees, which allow for complex decision-making processes. Zombies spawn in waves and prioritize attacking doors first. If a door is broken, they will infiltrate the base to attack players directly.

Weapons

The game features four different weapons: a pistol, shotgun, rifle, and grenade launcher. Each weapon has distinct ammo counts, damage values, area effects, and bullet spread rates.
